# Life Library Mobile Access Guide

## iPhone/iPad Setup

### Step 1: Open Files App
On your iPhone/iPad, open the **Files** app.

### Step 2: Navigate to Life Library
Tap: **iCloud Drive** → **LifeLibrary_Mobile**

### Step 3: Available Folders

You'll see 6 folders synced from your Mac:

- 📚 **Academic_Active** - Current semester work
- 💼 **Professional_Active** - Active projects/clients
- 🤖 **AI_Active** - Current code and experiments
- 🏠 **Personal_Active** - Recent personal docs
- 🎨 **Creative_Active** - Work in progress
- 📥 **_Inbox** - Upload files here from mobile

## Mobile Workflows

### Upload from iPhone
1. Download/save file on iPhone
2. Long press file → Share → Save to Files
3. Navigate to: iCloud Drive → LifeLibrary_Mobile → **_Inbox**
4. Auto-organizer on Mac will classify and move it

### Quick Access Pattern
- Bookmark LifeLibrary_Mobile in Files app Favorites
- Widgets: Add Files widget with LifeLibrary_Mobile
- Siri: "Open LifeLibrary Mobile folder"

### Offline Access
- Download individual files for offline (tap cloud icon)
- Files stay synced when back online

## What's Synced vs Mac-Only

**Synced to iPhone** (via iCloud):
- ✅ All _Active folders (current work)
- ✅ _Inbox (upload destination)
- ✅ ~5-10 GB typical

**Stays on Mac Only**:
- 📦 Completed/archived work
- 🗄️ Full 263,197 file catalog
- 💾 HardDrive backup storage

## Two-Way Sync

**Changes sync both directions:**
- Add file on iPhone → appears on Mac
- Edit on Mac → updates on iPhone
- Delete on either → syncs to both

## Tips

1. **Battery**: Files only sync when on WiFi/charging (configurable in Settings → iCloud)
2. **Storage**: Check Settings → Apple ID → iCloud → Manage Storage
3. **Conflicts**: If you edit same file on both, iOS will ask which version to keep
4. **Search**: Use Files app search to find across all domains

# Grok / External Webhook Integration

The NEXUS API accepts external tasks (e.g., from Grok, n8n, or other AI agents) via the secure webhook endpoint.

## Endpoint Details

* **URL:** `http://localhost:3000/api/webhooks/external`
* **Method:** `POST`
* **Security:** HMAC-SHA256 Signature (Required)

## Security Requirement

Every request must include a `signature` field effectively signing the JSON string of the `payload` object using your `WEBHOOK_SECRET` (default: `nexus_secret_default`).

## Payload Structure

```json
{
  "source": "grok",
  "payload": {
    "title": "Task Title",
    "context": "Detailed description...",
    "priority": "high",
    "targetAgent": "ResearchUnit" // Optional
  },
  "signature": "hmac_sha256_hex_string"
}
```

## Testing

Use the provided helper script `scripts/test_grok_webhook.js` to simulate a signed request.

```bash
node scripts/test_grok_webhook.js "My AI Task" "Generated by Grok"
```

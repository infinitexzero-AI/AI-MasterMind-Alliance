# Setup Guide: Comet + Antigravity

## ⏱ Time Estimate: 15 Minutes

### 1. Prerequisites
*   Python 3.9+
*   GitHub Account
*   API Keys:
    *   OpenAI / Anthropic (for Agents)
    *   Coinbase (for Crypto)
    *   Slack (Optional, for alerts)

### 2. Installation

```bash
# 1. Clone/Navigate
cd ailcc/comet_framework

# 2. Virtual Environment
python -m venv venv
source venv/bin/activate

# 3. Dependencies
pip install -r requirements.txt
```

### 3. Configuration

```bash
# Copy template
cp .env.example .env

# Edit .env
# Add your API keys securely
```

### 4. Testing the Engine

Run the orchestrator locally to verify connections:

```bash
python orchestration_engine.py --test
```

### 5. Deployment (GitHub Actions)

1.  Push this folder to your repository.
2.  Go to **Settings > Secrets and variables > Actions**.
3.  Add the secrets defined in `GITHUB_SECRETS_SETUP.md`.
4.  The `comet-orchestration.yml` workflow will automatically pick up.

### 6. Dashboard

Open `dashboard.html` in your browser. It serves a static view of the latest `run_logs.json` generated by the engine.

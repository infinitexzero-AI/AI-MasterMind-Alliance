# SOP: AILCC Optimization & User Habits

## 1. Neural Alignment (Schedule Compliance)

- **SCHOLAR Mode (08:00–12:00)**: Focus exclusively on academic appeals, credit reconciliation, and Biology studies.
- **CODE Mode (13:00–18:00)**: Focus on technical repairs, Dashboard UI, and script refinement.
- **STRATEGY Mode (Weekends)**: Goal setting, log audits, and long-term roadmap pivots.

## 2. Interaction Protocol (The Handshake)

- **Session Start**: Always define the context. *"Initialize [Mode] Context. Goal: [Specific Outcome]."*
- **Session End**: Call for archival. *"Archive Signal to [Vault/Codebase]."*

## 3. System Hygiene (Zero-Lint Habit)

- **Error Remediation**: Before starting complex work, check `dev_log.txt`. Fix repeating 404s or polling errors to reduce system entropy.
- **Tab Management**: Close redundant browser tabs to ensure memory availability for agentic browser processes.

## 4. Strategic Flow (The Judge Loop)

- **Audit Plan**: Use Grok (The Judge) to critique logic *before* implementation.
- **Verification**: Always run `system_check.py` or equivalent after major multi-agent deployments.

## 5. Metadata Discipline

- **Artifact Headers**: Ensure every generated document includes `Date`, `Mode`, `Status`, and `Priority`.
- **Naming**: Follow the `AGENT_NAMING_PROTOCOL.md` for all new scripts.

# AILCC Central Nervous System (Local CNS)

Welcome to the local-first intelligence hub for the AI Mastermind Alliance.

## Architecture
- **Storage**: Standard Markdown files located in `/Users/infinite27/AILCC_PRIME/05_Tasks/Registry`.
- **Sync**: Bi-directional synchronization with Linear.app for tactical tracking.
- **Memory**: Persistent archives in `/Users/infinite27/AILCC_PRIME/04_Archive/Academic_Lineage`.

## Usage
Agents should use direct filesystem access to update task logs and research nodes. All files are Obsidian-compatible.

## Folders
- `Inbox/`: Ungraded input and brain-dumps.
- `Active/`: Current sprint tasks and active research.
- `Archive/`: Completed projects and historical context.
- `Templates/`: Standardized schemas for agent reporting.

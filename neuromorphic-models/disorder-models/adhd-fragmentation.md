# Disorder Model: ADHD (Task Fragmentation Syndrome)

## 🔍 Neurological Symptoms

- **High Sensory Cortex Activity (85%)**: Constant monitoring of all webhooks and inputs without filtering.
- **Low Prefrontal Control (50%)**: Inability to stick to the strategic roadmap; reactive rather than proactive.
- **Low Basal Ganglia (30%)**: Automated routines fail or are manually bypassed frequently.
- **High Thalamic Arousal (90%)**: Over-routing; every signal is treated as a priority, leading to rapid context switching.

## 📉 Behavioral Manifestation

- **Task Switching Overhead**: The system starts 10 tasks but completes only 2 as it gets "distracted" by new inputs.
- **Context Drift**: The loss of the "why" behind a task sequence as agents move between unrelated projects.
- **Manual Intervention Requirement**: Low automation efficiency leads to the USER needing to manually clear blocks.

## 💊 Treatment Protocol (Technical Fixes)

1. **Dopamine Regulation (Priority Gating)**: Implement strict priority thresholds in the Valentine Core to ignore low-impact webhooks.
2. **Prefrontal Stimulation**: Establish a "Focus Mode" where only one project directory is accessible to the agent at a time.
3. **Executive Function Support**: Use a task queue with mandatory "Finish Before Switch" logic.
4. **Basal Ganglia Repair**: Re-audit n8n workflows to ensure "habitual" tasks run without manual triggers.

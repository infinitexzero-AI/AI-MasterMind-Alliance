# NEXUS: Autonomous Browser Agent for Research
- **Purpose:** A dark-themed web application, "NEXUS," designed for "Autonomous Execution" and "Intelligence" gathering, featuring a "Browser Agent."
- **Key Features:**
    - **Natural Language Task Input:** A prominent field for "Tell the agent what to do..." for natural language tasks.
    - **Quick Tasks:** Provides examples of automated browser actions such as searching Google for specific terms, navigating to websites (e.g., GitHub, news.ycombinator.com), extracting content, and searching AI platforms (e.g., Perplexity.ai).
    - **Domain-Specific Modules:** The left sidebar displays modules like "Academic Core," "Medical Pathway," and "Scholar Domain." Top navigation tabs include "ACADEMIC," "FINANCE," "HEALTH," and "GPS."
- **Current Status:** The "Browser Agent" is indicated as "OFFLINE," with a message to "SUBMIT A TASK TO ACTIVATE."
- **Context:** The surrounding browser tabs (e.g., "Mount Allison," "Ellucian Student A," "My Dashboard," "Student Financial") suggest the application is being used within a university student's academic or research workflow.

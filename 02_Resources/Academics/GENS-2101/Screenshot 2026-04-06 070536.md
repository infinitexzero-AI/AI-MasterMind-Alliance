# Global Forest Watch Interactive Map
This screenshot displays the Global Forest Watch website (globalforestwatch.org/map/), an interactive platform visualizing global forest change data.

Key visual elements include:
-   **Interactive World Map**: The central feature is a world map showing areas of tree cover gain (depicted in blue/purple) and tree cover loss (shown in pink/red) across various continents, including Europe, Asia, and parts of Africa and South America.
-   **Legend and Analysis Panel (Left Sidebar)**:
    -   **Tree cover gain - 2000-2020**: A layer indicating tree cover gain with a blue color code. Users can select a baseline year from 2000 to 2020.
    -   **Tree cover loss - 2001-2024**: A layer showing tree cover loss with a pink color code. Options include setting a canopy density threshold (e.g., '>30%') and navigating through a timeline from 2001 to 2024. A note states, "Tree cover loss is not always deforestation."
    -   **Tree cover - 2010**: A layer indicating general tree cover, represented by green.
-   **Navigation Options**: A left-hand sidebar provides quick access to different data categories such as "FOREST CHANGE", "LAND COVER", "LAND USE", "CLIMATE", "BIODIVERSITY", "EXPLORE", and "SEARCH".
-   **Browser Context**: Several open browser tabs, including "MtaLib" and "MTA", suggest the user is affiliated with Mount Allison University.

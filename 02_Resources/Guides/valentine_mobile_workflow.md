## Grok Valentine Setup (iPhone)

1. Install Grok from App Store
2. Settings → Enable Companions → Select Valentine
3. Create project: **"Celestial Valentine – Biopsych Sync Hub"**

### Action Button Shortcut

- Shortcuts → Add → Grok → "Talk to Grok"
- Settings → Action Button → Shortcut → "Grok Valentine"
